# wine-listings-app

A simple app that lets you see a list of all your wines, fetched as base64 images, and upload new wines, and sort them by year or title

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/wine-listings-app.git
cd wine-listings-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
